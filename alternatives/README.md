# Alternative Mockup Versionen

Diese alternativen HTML-Dateien demonstrieren verschiedene Implementierungsoptionen für die Gestaltungsentscheidungen des MVP.

## Axis 1: Learning Path Personalization

### lernpfad_option_a.html
**Option A:** Feste Sequenz, überspringe bestandene Module
- Module in fester Reihenfolge 1→2→3→4→5→6
- Keine Sub-Kompetenz-Filterung
- Einfach: "Test bestanden" für bestandene Module

### lernpfad_option_c.html
**Option C:** Vollständig dynamischer, LLM-generierter Lernpfad
- KI erstellt individuell optimierten Lernweg
- Kombiniert Inhalte über Kompetenzbereiche hinweg
- Passt sich kontinuierlich an Reflexionen und Fortschritt an
- Purple/AI-branded für visuelle Unterscheidung

## Axis 2: Didaktik-Methodenwahl

### lernpfad_chat_option_a.html
**Option A:** KI empfiehlt eine Methode
- Keine Auswahl zwischen mehreren Methoden
- KI wählt optimale Lernmethode basierend auf Profil
- Begründung der Empfehlung wird angezeigt

## Axis 3: Reflexion-Lernen Integration

### reflexion_chat_option_a.html
**Option A:** Unabhängig, keine Vorschläge
- Reiner Reflexionschat ohne Lernmodul-Verknüpfung
- Fokus auf Reflexion und Selbsterkenntnis
- Keine automatischen Empfehlungen

### reflexion_chat_option_c.html
**Option C:** Automatische Lernpfad-Anpassung
- Reflexionserkenntnisse passen Lernpfad automatisch an
- Live-Benachrichtigung über Anpassungen
- Neue Fokusmodule werden basierend auf Reflexion erstellt
- Purple/AI-branded für adaptive Funktionalität

## Verwendung für Präsentation

Diese Dateien sind für Screenshots gedacht und nicht in die Hauptnavigation des Mockups integriert. Verwenden Sie die entsprechenden relativen Pfade (`../styles.css`, etc.) um auf gemeinsame Ressourcen zuzugreifen.

**Aktuelle Mockup-Version (Option B für alle Achsen):**
- `/mockup/learning-path.html` - Assessment-basierte Priorisierung mit übersprungenen Sub-Kompetenzen
- `/mockup/learning-chat.html` - Lernende wählen frei zwischen Methoden
- `/mockup/reflexion_chat.html` - Kontextuelle Lernmodul-Vorschläge
